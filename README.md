Repositorio con el procesamiento y análisis estadístico del ramo de psicología "Instrumentos de medición para las Ciencias Sociales" de la Universidad de Chile, a cargo del profesor Daniel Miranda. Realizado en conjunto con Catalina Soza. 

En el curso, se realizó una operacionalización de un concepto para luego analizar su validez a través de diferentes análisis. En este caso, se realizaron análisis descriptivos, correlacionales, de fiabilidad (Alpha de Cronbach y Omega de McDonald) y Análisis Factorial Exploratorio. El grueso de los análisis fue conducido con la librería `psych` en el software estadístico R.
